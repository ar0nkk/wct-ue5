# Final Chase

中文 | [English](README.en.md)

基于 Unreal Engine 5.7 开发竞技跑酷追逐游戏，灵感来自 World Chase Tag

## 快速开始
```bash
git lfs install
git clone https://github.com/ar0nkk/wct-ue5.git
```
若要在一台设备上测试双人游戏，请打开项目后点击运行关卡按钮旁边的三个点（修改游戏模式和游戏设置），然后设置：
- 玩家数量 -> 2
- 网络模式 -> “以监听服务端运行”

## 游戏规则
1v1 竞技追逐游戏（追捕者 vs 逃跑者）

- **比赛结构**：每场比赛由 8 个回合组成
- **回合流程**：
  - **准备阶段（8秒）**：逃跑者可以自由在场地内移动选位；追捕者无法移动，但可以转动视角观察
  - **追逐阶段（20秒）**：追捕者恢复行动能力，开始抓捕逃跑者
- **胜负判定**：
  - **追捕者**：在 20 秒内触碰到逃跑者即获胜，回合立即结束但不得分
  - **逃跑者**：在 20 秒内未被触碰（存活），逃跑者获胜，并获得 1 分
- **角色轮换**：每回合结束后，胜者将在下一回合扮演**逃跑者**。8 回合全部结束后，得分较高者赢得整场比赛

## 操作方法 (更新中...)
- 目前使用局域网联机，主机进入游戏后，客户端输入主机的 IP 地址进行连接
- **WASD**：移动
- **Space**：跳跃
- **Shift**：冲刺
- **左键**：飞扑抓捕（动画待补充）
- **C**：低速-蹲伏；高速-滑铲
- **Ctrl**：行走

## 游戏特性
- **动作系统（GAS驱动）**：全套动作基于 Epic 的 Gameplay Ability System (GAS) 构建
- **飞扑**：追捕者可通过飞扑动作尝试触碰逃跑者，命中判定由服务器权威进行碰撞与角色验证
- **体力系统（开发中）**：
  - 角色具备独立的体力条
  - **消耗**：冲刺、跳跃、飞扑等高强度动作在不同程度上消耗体力
  - **回复**：未执行高消耗动作时，体力在常规状态下会缓慢恢复
