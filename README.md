# WCT (World Chase Tag)

Developed with Unreal Engine 5.7

## 游戏规则 (Game Rules)
1v1 竞技追逐游戏（追捕者 vs 逃跑者）

- **比赛结构（Match Structure）**：每场比赛由 16 个回合组成。
- **回合流程（Round Flow）**：
  - **准备阶段（8秒）**：逃跑者可以自由在场地内移动选位；追捕者无法移动，但可以转动视角观察。
  - **追逐阶段（20秒）**：追捕者恢复行动能力，开始抓捕逃跑者。
- **胜负判定（Win Conditions）**：
  - **追捕者（Chaser）**：在 20 秒内触碰到逃跑者即获胜，回合立即结束但不得分。
  - **逃跑者（Runner）**：在 20 秒内未被触碰（存活），逃跑者获胜，并获得 1 分。
- **角色轮换（Role Swap）**：每回合结束后，胜者将在下一回合扮演**逃跑者**。16 回合全部结束后，得分较高者赢得整场比赛。

## 游戏特性 (Features)
- **动作系统（GAS驱动）**：全套动作基于 Epic 的 Gameplay Ability System (GAS) 构建。
- **飞扑（Dive）**：追捕者可通过飞扑动作尝试触碰逃跑者，命中判定由服务器权威进行碰撞与角色验证。
- **体力系统（Stamina）**：
  - 角色具备独立的体力条。
  - **消耗**：冲刺（Sprint）、跳跃（Jump）、飞扑（Dive）等高强度动作在不同程度上消耗体力。
  - **回复**：未执行高消耗动作时，体力在常规状态下会缓慢恢复。

## 架构说明 (Architecture)
- **网络同步**：100% Server-Authoritative。所有回合流转、时间判定、分数计算和碰撞判定仅在服务端执行。
- **C++ 核心**：使用 `GameMode` 控制回合状态机，`GameState` / `PlayerState` 分发状态。
- **蓝图表现**：UI、特效与动画集成由派生的蓝图类完成。
